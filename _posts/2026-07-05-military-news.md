# 军事科技：AWS与Anduril推出战术边缘数据中心，DISA扩展JWCC云市场，EdgeCortix完成美国空军AI飞行测试

过去一周军事科技领域多项重要进展：AWS与Anduril合作推出基于AWS Outposts的战术边缘数据中心Menace-I，已在美国防部云市场上架；DISA计划通过统一云市场（UCM）大幅扩展JWCC生态，纳入更多SaaS供应商；日本EdgeCortix的SAKURA-II AI加速器完成美国空军飞行测试并获得DIU成功备忘录；AWS Project MAVERICK实战测试验证了对话式自主无人机蜂群控制和边缘AI处理能力。

## 主要新闻

### AWS与Anduril推出战术边缘数据中心Menace-I

AWS与Anduril于6月30日联合宣布推出面向国防的战术边缘解决方案，基于Anduril的Menace-I边缘平台和AWS Outposts构建。Menace-I提供三种变体：Core（边缘部署云，40-42台服务器）、Command（支持出击数据生成）和Enhanced（AI工厂，支持多达168台服务器，4个Outpost机架）。Menace已在过去三年中累计运行超过50,000小时的严酷现场使用，涵盖所有军种。该解决方案已在美国防部云市场上架，支持断网、中断、间歇和有限（DDIL）环境下的作战。Menace此前已托管Palantir的TITAN（陆军首个AI地面系统）软件。

**Source:** [AWS, Anduril Debut New Tactical Data Center Offering Listed on DOD's Cloud Marketplace](https://defensescoop.com/2026/07/01/aws-anduril-debut-new-tactical-data-center-cloud-offering/)

### DISA计划通过统一云市场扩展JWCC

DISA产品管理与发展主管John Hale和OCONUS云组合经理Adam Ringel在AWS Summit Washington D.C.上表示，DISA正在为JWCC准备重大重竞争——JWCC统一云市场（UCM）。UCM将超越核心超大规模基础设施，将软件即服务能力集成到同一生态系统中，创建一个国防部可以获取更广泛商业云服务的单一市场。在联合运营边缘（JOE）框架的多云架构内，DISA正在识别边缘所需的计算和AI能力以更好地支持作战人员。该计划反映了国防部从单一云采购向多云编排市场演进的战略转向。

**Source:** [DISA Plans Unified Cloud Marketplace to Expand JWCC](https://govciomedia.com/disa-plans-unified-cloud-marketplace-to-expand-jwcc/)

### EdgeCortix SAKURA-II完成美国空军飞行测试

日本EdgeCortix于7月3日宣布其SAKURA-II AI加速器平台成功完成美国空军大型部队演习中的飞行测试，并获得国防创新单元（DIU）成功备忘录——满足所有原型项目目标。该系统集成到原型高级智能网关系统中，以实现跨联合和联盟部队的弹性多域战场连接。SAKURA-II专为在尺寸、重量和功耗（SWaP）受限的战术环境中提供高性能、低功耗的AI推理能力而设计。这一成功标志着外国开发的边缘AI硬件首次在美国空军作战测试中通过验证。

**Source:** [EdgeCortix Puts Edge AI to the Test Aboard US Air Force Aircraft](https://thedefensepost.com/2026/07/03/edgecortix-air-force-flight-test/)

### AWS Project MAVERICK验证自主无人机蜂群与AI边缘处理

AWS Project MAVERICK于6月30日公布实战测试结果。MAVERICK是一个配备AWS Outposts、安全通信和灵活集成架构的移动加固平台。测试验证了在DDIL环境中通过自然语言对话控制自主无人机蜂群的能力。AWS与Anduril和Gambit合作，在真实作战环境下进行了多组织集成测试。Gambit开发的AMI在AWS区域完成测试后直接部署到野外Outposts上无缝运行，建立了云/边缘一致性的可重复模式。AWS表示MAVERICK将在2026年参与多次客户演习和演示。

**Source:** [Accelerating Autonomous System Innovation with Project MAVERICK Field Testing](https://aws.amazon.com/blogs/publicsector/accelerating-autonomous-system-innovation-with-project-maverick-field-testing/)

## 分析

AWS与Anduril的Menace-I合作是国防云市场格局的一个重要信号。AWS选择Anduril作为"首选边缘提供商"而非构建自有边缘硬件，表明AWS在战术边缘领域采取了合作伙伴生态优先的战略。Menace-I已在所有军种和作战行动（包括Epic Fury行动对伊朗）中证明了其实际运作能力。值得注意的是，Menace的Enhanced变体可以配置为AI工厂（最多168台服务器），这意味着AI推理能力正在被战术化——不仅在战略层面（如NATO Maven Smart System），也在旅级和营级作战单元层面部署AI能力。

DISA的JWCC统一云市场计划体现了国防部云采购逻辑的根本性转变。从单一超大规模云合同（如JEDI）到多供应商生态（JWCC），再到现在的"SaaS应用商店"（UCM），国防部正在将企业级云市场模式引入军事IT采购。这意味着像Palantir、Anduril这样的国防科技公司将可以通过市场模式直接向作战单元提供软件能力，而不需要总是通过大型系统集成商。这种去中介化可能显著加速新能力的部署速度。

EdgeCortix的SAKURA-II通过美国空军飞行测试是一个值得关注的非传统信号。一家日本公司的AI加速器在USAF大型部队演习中通过实战测试并获得DIU成功备忘录，表明美国国防部在AI硬件采购上正在超越传统供应商限制，以性能为首要标准。SAKURA-II在SWaP受限环境中的成功验证了边缘AI硬件市场正在成熟，对Nvidia在军事边缘AI推理市场的地位构成了潜在竞争。

Project MAVERICK的自然语言控制无人机蜂群展示了人机交互范式的演进方向。从预编程任务到摇杆控制，再到自然语言对话式指挥——作战人员可以用自然语言指令控制自主系统集群，这是一个质的飞跃。AWS的云/边缘一致性模式（在云中开发测试，在野外无缝运行）为国防软件开发提供了"dev/test once, deploy anywhere"的能力，这对降低军事软件的开发周期和成本具有重要意义。

## 结论

本周军事科技新闻呈现出三个清晰趋势：云计算正在从战略层面深入到战术边缘（Menace-I）；国防部IT采购从单一合同走向多元化市场生态（JWCC UCM）；边缘AI硬件和自主系统正在通过实战验证走向成熟（SAKURA-II、MAVERICK）。建议关注UCM的具体招标时间表和供应商入围情况，以及EdgeCortix是否会将SAKURA-II进一步推广至其他军种和盟国。
