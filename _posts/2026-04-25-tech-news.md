# 科技前沿：Google Cloud Next 2026全面拥抱AI Agent时代

本周Google Cloud Next 2026在拉斯维加斯盛大举行，超过32,000名行业领袖、开发者及合作伙伴参会。大会的核心主题是"Agentic Era"——AI Agent正在从根本上改变企业的工作方式。Google发布了全面的企业AI平台战略，展现了从芯片到应用层的垂直整合野心。

## 主要新闻

### Gemini Enterprise Agent Platform正式发布

Google正式推出Gemini Enterprise Agent Platform，这是对Vertex AI的重大升级和整合。该平台围绕四大支柱构建：构建、扩展、治理和优化。

**构建端**：引入了Agent Studio——一个用于使用自然语言创建Agent的低代码界面，以及升级的Agent Development Kit，包含用于编排多个协作Agent的新型图框架。

**运行时能力**：Agent Runtime提供亚秒级冷启动和秒级新Agent配置，支持长时运行Agent——可连续运行数小时甚至数天处理复杂业务工作流。Memory Bank功能赋予Agent跨会话的持久长期记忆。

**治理与安全**：Agent Identity、Agent Gateway和Agent Registry帮助IT团队追踪Agent身份并执行AI防护栏。Agent Evaluation、Agent Observability和Agent Simulation在部署后评估Agent行为并测试其对合成用户提示的响应。

**Source:** [Google expands Gemini Enterprise](https://www.itpro.com/technology/artificial-intelligence/google-expands-gemini-enterprise-consolidates-vertex-ai-services-to-simplify-agent-deployment)

### 第八代TPUs：TPU 8t和TPU 8i

Google发布了第八代Tensor Processing Units，包括：

**TPU 8t（训练）**：配备TPU Direct Storage和Google Cloud的Virgo Networking，采用Boardfly架构进行大规模Agent扩展，以及Collectives Acceleration Engine，可降低延迟和功耗。

**TPU 8i（推理）**：专门优化用于推理工作负载。

公司还宣布了AI优化网络架构，支持250,000节点RDMA虚拟私有云。

**Source:** [Google Cloud Agentic Data Cloud](https://www.constellationr.com/insights/news/google-cloud-presses-full-stack-ai-edge-new-tpus-agentic-data-cloud-gemini-enterprise)

### Agentic Data Cloud：面向Agent时代的数据架构

Google Cloud推出了Agentic Data Cloud，这是一种专为AI时代设计的新数据平台。其核心组件包括：

**Knowledge Catalog**：为AI Agent工具提供适当的上下文环境。

**跨云数据湖屋**：使AI工具能够访问AWS、Microsoft Azure以及Google Cloud中的数据。

**Data Agent Kit**：开源工具，包含Model Context Protocol工具、环境特定扩展和预构建插件，使开发者能够在VS Code和Claude Code等平台中使用Google的Agent开发工具。

**Source:** [Google unveils data cloud purpose built for agentic AI](https://www.techtarget.com/searchdatamanagement/news/366641929/Google-unveils-data-cloud-purpose-built-for-agentic-AI)

### 企业采用率数据

Google披露，75%的Google Cloud客户正在使用其AI产品。过去12个月中，330个Google Cloud客户处理了超过一万亿个token。API直接使用模型现在每分钟处理超过160亿个token，较上季度增长60%。

**Source:** [Google Cloud Next 2026: News and updates](https://blog.google/innovation-and-ai/infrastructure-and-cloud/google-cloud/next-2026/)

### 资本支出承诺

CEO Sundar Pichai在主题演讲中重中了Alphabet今年的资本支出计划：1750亿至1850亿美元。其中，"略过一半"的机器学习计算能力投资将专门用于云业务。

**Source:** [Google puts AI agents at heart of its enterprise money-making push](https://www.reuters.com/business/google-puts-ai-agents-heart-its-enterprise-money-making-push-2026-04-22/)

## 分析

**垂直整合战略**：Google的AI战略明显倾向于垂直整合——从芯片(TPU)、基础设施、网络，到模型、Agent框架和应用层。这种整合在理论上可以产生更好的整体结果，因为各层可以共同优化。与组装来自不同供应商的组件相比，这种方法在性能调优和成本优化方面可能更具优势。

**Agent管理的企业需求**：随着企业开始部署AI Agent，治理和可见性成为关键需求。Gemini Enterprise Agent Platform通过Agent Identity和Agent Gateway等产品直接回应了这一需求。这表明AI Agent不仅仅是技术问题，更是一个管理和治理问题。

**数据架构的根本性变革**：Agentic Data Cloud的发布反映了企业数据基础设施的根本性变革需求。传统的数据架构是为人类设计的——供人类查询和分析。而新的架构是为Agent设计的——允许AI系统自主访问和操作数据。这种转变需要重新思考数据治理、访问控制和元数据管理。

**竞争格局**：微软、亚马逊AWS和Salesforce最近都进入了Agent编排和管理领域。Google的差异化在于其端到端整合能力和TPU硬件优势。然而，企业客户可能更倾向于避免供应商锁定，这为多供应商策略留下了空间。

**市场信号**：75%的Google Cloud客户使用AI产品，以及API使用量的60%季度增长，都表明企业AI采用正在进入主流。Google押注Agent作为下一个增长引擎，这一赌注能否成功将取决于企业采纳速度和实际业务价值实现。

## 结论

Google Cloud Next 2026清晰地展示了AI企业应用的演进方向——从单纯的模型托管到完整的Agent平台。260项发布中，Gemini Enterprise Agent Platform和Agentic Data Cloud是最具战略意义的。对于企业而言，关键是评估这些能力如何与现有技术栈集成，以及如何建立有效的Agent治理框架。随着AI Agent开始承担越来越复杂的业务工作流，可观察性、安全性和合规性将成为必须解决的问题。
