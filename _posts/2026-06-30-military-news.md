# 军事科技：空军授予CCA生产合同，Army选定Anduril主导NGC2数据层，Pentagon推出JWCC云市场

美国国防部在无人自主系统、指挥控制和云计算基础设施三大领域同时取得重大进展，标志着军事数字化转型进入加速部署阶段。

## 主要新闻

### 美国空军正式授予General Atomics和Anduril CCA生产合同

美国空军正式选定General Atomics（FQ-42A）和Anduril Industries（FQ-44A）建造首批至少150架协作作战飞机（CCA），比预定时间提前四个月。同时，Anduril、Shield AI和Collins Aerospace获得自主飞行软件竞争合同，将经过两轮六个月竞争后在2027年夏季确定主软件供应商。空军强调"任务自主性是CCA概念的基石"，采用的政府自主参考架构（A-GRA）实现了软件与硬件的解耦，使自主软件可在不同飞机间移植和更新。

**Source:** [US Air Force awards first CCA production contracts to General Atomics, Anduril](https://www.defensenews.com/industry/techwatch/2026/06/18/us-air-force-awards-first-cca-production-contracts-to-general-atomics-anduril/)

### Army选定Anduril主导下一代指挥控制通用数据层

美国陆军宣布确立Next Generation Command and Control（NGC2）通用数据层基线，选定Anduril Industries领导该计划。Anduril将保持与Palantir的合作伙伴关系，提供基于Lattice和Foundry的边缘到云端数据网格，并与Raft合作进行数据注册、转换工具和数据联邦。Lockheed Martin将继续领导25th步兵师的"全栈"实施。该决定基于4th ID和25th ID在Ivy Mass和Lightning Surge演习中十个月的反馈和数据积累。7月的Project Convergence-Capstone 6将是师级NGC2验证的压轴活动。

**Source:** [Army taps Anduril as lead for NGC2 common data layer baseline as service sets 'groundwork for rapid scaling'](https://defensescoop.com/2026/06/22/army-taps-anduril-lead-ngc2-common-data-layer-baseline/)

### Pentagon JWCC后续合同将创建云市场，扩展AI和边缘计算

国防信息系统局（DISA）发布JWCC Unified Cloud Marketplace（UCM）草案工作说明书。新合同将建立三级市场结构：超大规模云服务商（第一级）、Everything-as-a-Service（第二级）和创新型小企业（第三级）。重点包括：在拒绝、干扰、间歇或有限带宽（DDIL）环境中部署云能力；纳入边缘设备和边缘服务；要求云服务商在所有安全级别提供AI/ML能力。合同预计2027年开始授予。

**Source:** [Pentagon's JWCC follow-on would create cloud marketplace, expand AI and edge computing](https://defensescoop.com/2026/06/01/pentagon-jwcc-ucm-draft-performance-of-work-statement/)

### 海军向AWS、Azure、GCP和Oracle授予JWCC任务订单

海军Neptune云管理办公室宣布向Amazon Web Services、Microsoft Azure、Google Public Sector和Oracle America授予JWCC任务订单。Nepton负责集中管理海军云服务采购，通过每个CSP的专用任务订单实现集中化成本管理并利用大合同折扣。扩大云服务商组合将为海军和海军陆战队任务所有者提供更广泛的能力目录，同时加速数字转型。

**Source:** [Department of the Navy announces cloud computing task orders for Joint Warfighting Cloud Capability (JWCC) contract](https://www.dvidshub.net/news/567917/department-navy-announces-cloud-computing-task-orders-joint-warfighting-cloud-capability-jwcc-contract)

### Argonne衍生公司获得五角大楼混合云-超算合同

由Argonne国家实验室孵化的Parallel Works获得五角大楼高性能计算现代化计划（HPCMP）合同，提供统一平台连接军事超算中心与安全商业云基础设施。其ACTIVATE High Security Platform已获Impact Level 5认证，是少数获准处理出口管制工作负载（包括ITAR和CUI）的软件平台之一。用户可通过单一界面访问AWS、Azure、Google Cloud和Oracle Cloud基础设施。

**Source:** [Argonne spinout wins Pentagon deal to unify cloud and supercomputing](https://interestingengineering.com/military/pentagon-parallel-works-hybrid-cloud-ai-supercomputing)

## 分析

本周的军事科技新闻揭示了美国国防部数字化转型的三大支柱同步推进。首先，无人自主系统正从实验快速进入量产阶段。CCA生产合同提前四个月授予General Atomics和Anduril，表明空军对"忠诚僚机"概念已从"是否可行"转向"如何快速部署"。A-GRA架构实现的软硬件解耦是关键设计决策——它使自主软件可在不同机型间移植，避免被单一供应商锁定。

其次，陆军NGC2项目的进展标志着指挥控制体系架构的根本转变。选定Anduril的Lattice作为通用数据层基线意味着陆军正在从传统的平台中心C2模式转向数据中心C2模式。Lattice的边缘到云端数据网格架构使AI模型和应用程序可在战场前沿和战略后方之间无缝流动。PC-C6在7月的师级验证将是检验这一新架构的关键测试。

第三，云计算基础设施的统一和扩展正在为上述两部分提供基础。JWCC UCM的三级市场结构和海军的Neptune任务订单说明国防部正在系统性地构建一个多供应商、多安全级别的"军事云生态"。Parallel Works的混合云-超算合同则填补了高性能计算与商业云之间的空白。

值得关注的是Anduril出现在三个独立的重大项目中——CCA生产、CCA自主软件竞争和NGC2数据层。这表明这家公司已经成为五角大楼在自主系统和软件定义军事能力方面的核心合作伙伴。

## 结论

2026年6月标志着美国国防部数字化转型从概念验证进入规模化部署的转折点。CCA量产、NGC2数据层确立和JWCC云市场建设三者同步推进，展示了无人系统、数据架构和云计算基础设施的深度融合。对于国防工业和盟友而言，理解这一转型的技术架构（A-GRA、Lattice、JWCC）比跟踪单个合同更为重要——这些架构选择将定义未来十年的军事技术格局。
