# 科技技术动态：DeepMind破解56年数学难题与AWS-Google云直连免流量费

本周科技领域多个方向同时发力：Google DeepMind的AlphaProof Nexus独立解决9个Erdős难题——其中一些已悬而未决56年；AWS推出与Google Cloud的直连接口并取消数据传输费；Alibaba Cloud在新加坡发布Qwen 3.7-Max旗舰模型；Google SynthID水印技术获得NVIDIA和OpenAI采用，并预览内容检测API。

## 主要新闻

### DeepMind AlphaProof Nexus独立证明9个Erdős猜想

Google DeepMind宣布其AI系统AlphaProof Nexus自主解决了9个由Paul Erdős提出的开放数学问题，包括一些已存在56年未获证明的难题。系统还证明了44个OEIS中的开放猜想，解决了代数几何中一个15年的问题，并发现了优化理论中此前未被人类发现的算法参数。系统基于Gemini 3.1 Pro驱动推理，通过Lean形式化验证系统自动检查每一步逻辑。但DeepMind CEO Demis Hassabis明确表示，这"远未达到"真正的AGI，强调通用智能需要跨领域的广泛原创能力。

**Source:** [DeepMind's Math AI Solves 56-Year Erdős Problems](https://www.qoo10.co.id/en/tech/103308/deepminds-math-ai-solves-56-year-erdos-problems-yet-hassabis-says-it-is-not-agi/)

**Source:** [Google DeepMind AlphaProof Nexus](https://deepmind.google/discover/blog/alphaproof-nexus/)

### AWS与Google Cloud直连接口GA：取消数据传输费

AWS Interconnect多云服务正式GA，提供Amazon VPC与Google Cloud VPC之间的私有Layer 3直连（不经过公网），且取消了按GB计费的数据传输费。用户只需按预留带宽支付固定费用——10Gbps同区域约为每小时12.33美元（月均约9,000美元，无传输上限），远低于传统跨云数据传输成本。Azure和Oracle Cloud的支持将于2026年下半年上线。AWS还发布了Connection Coordinator API规范（开源GitHub），邀请区域性云服务商接入。

**Source:** [AWS Connected Its Network to Google Cloud With No Egress Fees](https://webhosting.today/2026/05/22/aws-connected-its-network-to-google-cloud-with-no-egress-fees-azure-and-oracle-are-next/)

### Alibaba Cloud发布Qwen 3.7-Max：万亿参数Agent模型

Alibaba Cloud在新加坡首届Qwen国际大会上发布旗舰模型Qwen 3.7-Max，拥有超过1万亿参数和100万token上下文窗口，专为Agent工作流设计。同时推出Qwen Cloud AI原生平台和新技能门户，支持60+云产品转换为Agent可用技能。公司还与NTUC和ST Telemedia合作，计划培训1,000家新加坡中小企业掌握生成式AI技能。Qwen 3.7-Max体现了从"知识摘要工具"到"数字劳动力"的AI范式转变。

**Source:** [Alibaba unveils Qwen 3.7 Max at inaugural Singapore conference](https://www.computerweekly.com/news/366643330/Alibaba-unveils-Qwen-37-Max-at-inaugural-Singapore-conference)

### Google SynthID获NVIDIA、OpenAI采用，内容检测API预览

Google宣布SynthID AI内容水印技术被NVIDIA（整合至Cosmos基础模型）和OpenAI（结合C2PA元数据采用多层验证）采用。同时预览SynthID Content Detection API，支持JPEG/PNG/WebP图片通过REST API提交并检测AI合成内容。Google已为超过1,000亿张图片/视频和6万小时音频添加SynthID水印。该API首批合作伙伴包括Shutterstock、Snap、Fox Sport、Canva等。

**Source:** [Google Expands SynthID Adoption for AI Watermarking](https://www.infoq.com/news/2026/05/google-synthid-content-detection/)

### AWS MCP Server正式GA：服务编排新范式

AWS的托管MCP Server正式GA，暴露所有AWS API通过标准语言无关接口，支持IAM驱动的访问控制、CloudWatch指标和CloudTrail审计日志。AI编码代理（如Claude Code、Cursor）现在可以无需长期凭证即可调用AWS服务。与Azure OpenAI Functions和Google Vertex AI Agents相比，AWS MCP在服务覆盖范围上领先。AWS的轻量级Python沙箱减少了冷启动延迟。

**Source:** [AWS MCP Server Goes GA](https://news.lavx.hu/article/aws-mcp-server-goes-ga-what-it-means-for-multi-cloud-ai-agent-strategies)

## 分析

本周全球科技新闻呈现三个并行趋势：AI能力的边界拓展、云基础设施的互联互通、以及AI治理和认证体系的建立。

DeepMind的数学突破具有特殊意义——它不是在更大数据上训练更大模型，而是通过形式化验证（Lean）实现了从"统计模式匹配"到"逻辑推理保证"的跃升。这可能是通往可靠AI的关键路径。Hassabis克制地否认AGI的说法，恰恰体现了一种负责任的科学态度——将成果限定在可验证的范围内。

AWS与Google Cloud的直连接口取消数据传输费，是云计算产业的里程碑事件。多云架构此前因高昂的出口流量费而难以经济地实施，这一变动将释放大量被锁定在单一云上的企业工作负载。AWS开放Connection Coordinator API的举动，则可能催生新一代多云网络生态。

SynthID的生态化扩张——NVIDIA和OpenAI同时采用——表明AI内容认证正在从单个公司的解决方案走向行业标准。与C2PA元数据的多层互补方案，可能在2026年下半年成为AI生成内容验证的事实标准。

## 结论

本周科技发展呈现"数学推理突破、基础设施互联、内容认证标准化"三条主线。DeepMind的形式化验证方法可能成为下一代AI可靠性的关键；AWS-GCP直连将加速企业多云战略落地；SynthID的生态化则为AI治理提供了可操作的基础设施。企业应同时关注这三大趋势，在规划2026下半年技术路线时纳入考量。
